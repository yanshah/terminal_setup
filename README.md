# Terminal Setup

This repository provides a script to set up iTerm2 with Zsh, including configuration files and plugins, on a new macOS machine.

## Features
- iTerm2 settings
- Zsh configuration
- Zsh plugins:
  - zsh-syntax-highlighting
  - zsh-autosuggestions
  - powerlevel10k theme
